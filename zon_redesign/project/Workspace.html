Key rules

✅ @riverpod codegen — run dart run build_runner build after model/provider changes
✅ setState() only for local ephemeral UI state (tabs, toggles, form controllers)
✅ context.push() for drill-down, context.go() for tab navigation
✅ Minimum tap target: 44×44pt on every interactive element
✅ const constructors wherever possible
❌ No business logic in widgets
❌ No direct Supabase calls in widgets
❌ No fixed-height text containers that clip at 1.5× font scale
❌ Never add dependencies without updating pubspec.yaml

Provider / state pattern

feedNotifierProvider — stamp list, pagination, like/save toggle
feedStoriesProvider — CheckInStory list for stories rail
gpsNotifierProvider — sessionPath, live position
timelineNotifierProvider — keepAlive, DayBundle (checkIns + stamps + notes + diary)
profileNotifierProvider(id) — UserProfile
profileStampsNotifierProvider(id, publicOnly) — stamp grid
followStateProvider(id) / friendStateProvider(id)
ghostModeProvider — from LocationSharingRepository
friendLocationsProvider — Realtime FriendLocation stream
unreadNotificationCountProvider — bell badge count
